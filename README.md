# TRPG三折页排版器

一个用于制作TRPG（桌上角色扮演游戏）规则三折页的单页Web应用工具。

## 功能特点

### 1. 页面设置
- 可自由调整页面宽度和高度（单位：毫米）
- 支持自定义页面数量（1-10页）
- 默认A4纸张大小（210x297mm）

### 2. 内容块
提供多种类型的内容块：
- 📝 文本块：标题+正文
- 🏷️ 标题块：大标题
- 🖼️ 图片块：支持上传本地图片
- 📊 表格块：支持JSON格式编辑
- 💬 引用块：引用样式文本

### 3. 编辑功能
- 点击任意内容块即可在侧边栏编辑
- 支持编辑标题、内容
- 可调整字体大小（8-72px）
- 支持左对齐、居中、右对齐
- 可拖拽排序内容块
- 支持在不同页面之间移动内容块

### 4. 导出功能
- 保存到本地（JSON格式）
- 从本地文件加载
- 导出为HTML文件（支持单页或全部导出）
- 导出为PDF文件（支持单页或全部导出）

## 使用方法

### 本地运行
1. 直接在浏览器中打开 `index.html` 文件
2. 开始创作你的TRPG三折页！

### 部署到GitHub Pages
1. 访问新的GitHub仓库：[https://github.com/jeffdyuyi/threepages](https://github.com/jeffdyuyi/threepages)
2. 将这四个文件上传到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源
5. 稍等几分钟，你的应用就可以在线访问了！
